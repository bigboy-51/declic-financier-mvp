9. **Data Flow**
   
   In Charges component:
```typescript
   1. Fetch: users/{uid}/charges/ from Firebase
   2. Loop: CHARGE_CATEGORIES.map()
   3. For each category: Get rubriques array
   4. Pass to ChargeCard component
   5. ChargeCard displays card + rubriques
   6. User interactions (edit/delete) → PROMPT 3
```
   
   Pseudo-code:
```typescript
   const ChargesTab = () => {
     const [charges, setCharges] = useState({});
     
     useEffect(() => {
       // Fetch charges from Firebase
       const chargesRef = ref(database, `users/${uid}/charges`);
       onValue(chargesRef, (snapshot) => {
         setCharges(snapshot.val() || {});
       });
     }, [uid]);
     
     return (
       <div className="charges-tab">
         {CHARGE_CATEGORIES.map(category => (
           <ChargeCard
             key={category.id}
             category={category}
             rubriques={charges[category.id]?.rubriques || {}}
             onAddRubrique={handleAddRubrique}
           />
         ))}
       </div>
     );
   };
```
