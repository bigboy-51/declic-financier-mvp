2. **Firebase Structure - Personal Profile Data (Per profile)**
   
   users/{uid}/boss_profile/ (PERSONAL)
   ├── quizCompleted: true/false
   ├── profileType: "strategiste" | "economiste" | "plaisancier" | "chercheur" | "penseur"
   ├── profileDescription: "Vous avez une vision long terme"
   ├── quizCompletedAt: ISO timestamp
   ├── quizAnswers: [0, 1, 2, 3, 4] (answer indices Q1-Q5)
   └── quizEscalation/
       ├── escalationLevel: 1 | 2 | 3
       ├── skipCount: 0-3
       ├── lastSkippedAt: ISO timestamp
       ├── userContinuedWithoutQuiz: true/false
       └── quizCompleted: true/false
   
   users/{uid}/wife_profile/ (PERSONAL - same structure)
   ├── quizCompleted: false
   ├── profileType: null
   ├── quizEscalation/
   │   ├── escalationLevel: 1
   │   ├── skipCount: 0
   │   └── ...
   └── ...
   
   (For single user: use "single_profile" instead)

3. **Data Persistence - Implementation**
   
   STEP 1 (Initial Balance):
   - On complete: Save to Firebase
   - Path: users/{uid}/finances/initialBalance = amount
   - Also save: users/{uid}/finances/currentBalance = amount
   - Also save: users/{uid}/finances/lastUpdated = ISO timestamp
   - Mark: users/{uid}/onboarding/step1_completed = true
   - Check: If all steps done → set onboarding/completed = true
   
   STEP 2 (Debts):
   - On add debt: Save to Firebase
   - Path: users/{uid}/debts/debt_[uniqueId]/
   - Include: name, monthlyPayment, remainingBalance, createdAt
   - On delete debt: Remove from Firebase
   - On edit debt: Update in Firebase
   - Mark: users/{uid}/onboarding/step2_completed = true (after first debt)
   
   STEP 3 (Charges):
   - On add expense: Save to Firebase
   - Path: users/{uid}/charges/[category]/charge_[uniqueId]/
   - Include: name, budgeted, actual, createdAt
   - On edit: Update in Firebase
   - On delete: Remove from Firebase
   - Mark: users/{uid}/onboarding/step3_completed = true (after first expense)
   
   STEP 4 (Savings Goals):
   - On add goal: Save to Firebase
   - Path: users/{uid}/savingsGoals/[timeframe]/goal_[uniqueId]/
   - Include: name, targetAmount, currentSaved, timeline, createdAt
   - On edit: Update in Firebase
   - On delete: Remove from Firebase
   - On reach 100%: Mark as completed, show celebration "🎉 Objectif atteint!"
   - Mark: users/{uid}/onboarding/step4_completed = true (after first goal)
   
   STEP 5 (Couple - Invite Partner):
   - On invite sent: Save to Firebase
   - Path: users/{uid}/couple/inviteCode = [code]
   - Mark: users/{uid}/onboarding/step5_completed = true
   
   Quiz Results (Per profile):
   - On quiz complete: Save to Firebase
   - Path: users/{uid}/[currentProfile]/
   - Save: quizCompleted = true
   - Save: profileType = result (strategiste, etc)
   - Save: profileDescription = description text
   - Save: quizCompletedAt = ISO timestamp
   - Save: quizAnswers = [answers array]
   
   Quiz Escalation (Per profile):
   - On skip: Update Firebase
   - Path: users/{uid}/[currentProfile]/quizEscalation/
   - Update: skipCount++
   - Update: lastSkippedAt = ISO timestamp
   - Update: escalationLevel = 1/2/3
   - On completion: Set quizCompleted = true

4. **Data Retrieval - On App Load**
   
   When user logs in:
   1. Get currentUser.uid
   2. Get currentProfile from localStorage (or show selection)
   3. Load users/{uid}/onboarding/ → Show which steps completed
   4. If onboarding/completed = true → Load financial data:
      - Load users/{uid}/finances/ → Display solde on Dashboard
      - Load users/{uid}/debts/ → Display Credits tab
      - Load users/{uid}/charges/ → Display Charges tab + Budget
      - Load users/{uid}/savingsGoals/ → Display Savings tab + Progress
   5. Load users/{uid}/[currentProfile]/ → Load personal profile data:
      - Load quizCompleted → Decide if quiz modal needed
      - Load profileType → Apply personalization to dashboard
      - Load quizEscalation → Check escalation level
   6. If quiz not completed:
      - Check escalationLevel
      - Show appropriate escalation modal (1, 2, or 3)

7. **Single vs Couple - Profile Path**
   
   SINGLE USER:
   - Profile data: users/{uid}/single_profile/
   - Quiz: users/{uid}/single_profile/quizCompleted
   - Escalation: users/{uid}/single_profile/quizEscalation/
   
   COUPLE USER:
   - Profile data: users/{uid}/boss_profile/ + users/{uid}/wife_profile/

   - Quiz (wife): users/{uid}/wife_profile/quizCompleted
